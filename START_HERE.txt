   💡 TROUBLESHOOTING:
   
   Issue                        Solution
   ──────────────────────────── ─────────────────────────────────
   Game won't load             Check http://localhost:3000
   Car won't move              Use arrow keys (← →)
   ESP8266 not connecting      Check WiFi (must be 2.4GHz)
   MQTT connection failed      Credentials are pre-configured
   Gestures too sensitive      Adjust X_THRESHOLD in .ino file
